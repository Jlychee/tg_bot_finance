# tg_bot_finance

*Telegram Finance Tracker* — это бот для учёта личных финансов. Он 
позволяет пользователю добавлять доходы и расходы прямо в Telegram, а 
все данные автоматически сохраняются в базе Notion.

Бот поддерживает удобные команды, выбора кошелька, добавление расходов/доходов/переводов и интеграцию через Notion API.
